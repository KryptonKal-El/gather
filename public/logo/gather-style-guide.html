04 — Guidelines
Usage Notes

✅ Do

  • Use Forest (#3D7A63) for the wordmark on light backgrounds
  • Use White wordmark on the Forest dark background
  • Always include the 🩷 heart on the app icon
  • Maintain the gradient direction (135°) on the icon
  • Keep Nunito 800 for all logo text
  • Use Cream (#FAFAF7) as the default app background

❌ Don't

  • Don't stretch or distort the logo proportions
  • Don't place the logo on busy or clashing backgrounds
  • Don't use colors outside the defined palette
  • Don't change the font family away from Nunito
  • Don't use the icon without its rounded-square container
  • Don't alter the heart accent color

🎯 Logo Placement

  • App Store / Play Store → Icon only
  • App navigation bar → Icon + Name (horizontal)
  • Website header → Full logo with tagline
  • Splash / onboarding → Stacked centered version
  • Social media profile → Icon only (square crop)
  • Email footer → Icon + Name (horizontal)

📐 Spacing & Radius

  • App icon corner radius: 28% of icon width
  • Card corner radius: 16px standard, 24px large
  • Clear space around logo: 1× icon height minimum
  • Button border radius: 12px
  • Base spacing unit: 8px grid
  • Shadow: 0 4px 16px rgba(61,122,99, 0.12)
