# iOS Widgets Progress

## Codebase Patterns

- **Widget data sharing:** Main app writes to `SharedDataStore` (App Group container), widgets read via `WidgetDataProvider`
- **Auth in widgets:** `SharedDefaults` stores Supabase URL, anon key, and auth tokens; `WidgetSupabaseClient` uses REST API (not full SDK) to keep binary small
- **Widget refresh:** Call `WidgetCenter.shared.reloadAllTimelines()` after any SharedDataStore write
- **Deep links:** Use `gatherlists://list/{listId}` for widget tap navigation; handled in `GatherListsApp.onOpenURL`
- **Lock screen widgets:** Use `.accessoryCircular` and `.accessoryRectangular` families with `.containerBackground(.fill.tertiary, for: .widget)`
- **Configuration intents:** `SelectListIntent` for single-list widgets, `SelectListOrAllIntent` for due items (supports "All Lists" option)
- **Image picker view mode:** Use `@State private var viewMode: Bool` initialized from `item.imageUrl` presence to toggle between view-only (large image + remove) and picker (tabs + search + upload) modes
