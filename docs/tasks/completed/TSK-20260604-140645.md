# TSK-20260604-140645: Redesign hero section to center text/CTA inside iPhone mockup image

## Request
We didn't finish the marketing page hero section, we need to add this iPhone image as a background behind the hero text & button (which these need to be centered inside of this iPhone image)

## Investigation Findings

### Recommended Approach
- Replace the current split hero layout (text left / CSS phone right) with a single centered layout where the user-provided iPhone image is the focal element
- Use `position: relative` on a `.hero-phone-stack` container holding both the iPhone `<img>` and a `.hero-phone-content` overlay div with `position: absolute`
- Center the overlay text/CTA within the iPhone screen area using percentage-based insets (~4–5% from each edge to account for the metal frame border)
- Keep the existing particle canvas animation as the hero section background — particles visible around the phone frame look great on the dark bg
- Remove the old pure-CSS iPhone mockup (`.phone-frame`, `.phone-notch`, `.phone-screen`) — replaced by the real image
- Switch App Store badge to the white variant since the phone screen is dark — black badge would be nearly invisible
