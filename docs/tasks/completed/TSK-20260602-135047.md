# TSK-20260602-135047: Manage Stores — tappable rows with chevron and × close button

## Request
Each store row should have a ">" chevron and be tappable to open its edit page.
Add an × icon close button in the top-left so users have an explicit way to dismiss the sheet.

## Investigation Findings

### Recommended Approach
- Wrap `storeRow` in a `Button` that sets `storeToEdit = store`, replacing the context menu "Edit Name & Color" trigger (context menu can stay for discoverability but tap is primary)
- Add `Image(systemName: "chevron.right")` trailing in the `storeRow` HStack, styled `.foregroundStyle(.tertiary)` and `.font(.caption)` to match standard iOS list disclosure indicators
- Add `@State private var dismiss` environment value and a leading `ToolbarItem(placement: .topBarLeading)` with an `xmark` button

### Confidence
HIGH

### Evidence
- [CONFIRMED] `storeRow` is a plain HStack with no tap gesture or button wrapper (code trace)
- [CONFIRMED] `storeToEdit` state + `.sheet(item: $storeToEdit)` already exists — just needs a new trigger (code trace)
- [CONFIRMED] No `@Environment(\.dismiss)` exists in `StoreBrowserView` today (code trace)
- [CONFIRMED] Toolbar already has leading `EditButton` (conditional) and trailing `+` button — xmark fits leading slot (code trace)

### Eliminated Approaches
- `NavigationLink` — unnecessary since `EditStoreSheet` is a sheet, not a push navigation destination

### Scope
Small (1 file, ~15 lines changed)
